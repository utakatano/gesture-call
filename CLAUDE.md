このファイルは、このリポジトリでコードを扱う際のClaude Code (claude.ai/code) へのガイダンスを提供します。

## 言語設定

ドキュメントとClaude Codeのメッセージ・レスポンスは日本語を使用してください。

## プロジェクト概要

Gesture Callは、Webカメラを通じて手の動きを検知し、アクションをトリガーするジェスチャー認識システムです。プロジェクトは2つのフェーズに分かれています：

1. **Webアプリケーション**: カメラ映像を表示し、ジェスチャー（手を挙げる、指一本、親指を上げる）を検知するシンプルなWebページ
2. **Chrome拡張機能**: ポップアップコントロールとオーバーレイカメラ表示を持つブラウザ拡張機能

## アーキテクチャ目標

- **CSP準拠**: Chrome拡張機能のContent Security Policy制約内で動作するコード
- **リアルタイム処理**: ジェスチャー検知のためのWebカメラ映像処理
- **デュアルプラットフォーム**: Webアプリケーションと拡張機能間で共有されるコアジェスチャー検知ロジック

## プロジェクト構造

リポジトリは現在、初期計画段階にあります。`docs/tasks.md`の要件に基づく、予想される構造は以下を含みます：

- ローカルサーバーを持つWebアプリケーション
- manifest、popup、content scriptsを持つChrome拡張機能
- 共有ジェスチャー検知モジュール
- カメラハンドリングコンポーネント
